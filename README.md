#openrun
당신 만의 캐릭터, 함께 달리는 즐거움!

프로젝트 소개
언제든지 다양한 사람들과 달리기 모임에 참여하고, NFT 아바타를 획득하여 나만의 개성을 표현할 수 있는 커뮤니티 기반 '함께 뛰는 M2E' dApp 입니다.
